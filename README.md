# Barrage-helper

> 一个弹幕助手

使用 electron-vue 开发的上一版本见分支 [with-electron-vue](https://github.com/WozHuang/Barrage-helper/tree/with-electron-vue)

## Feature

- [ ] 弹幕播报/礼物播报
- [ ] 窗口置顶
- [ ] 窗口透明/背景色切换
- [ ] 全屏弹幕
- [ ] 实现语音播报
- [ ] 自动升级
- [ ] 外部全屏
- [ ] 模拟 switch 外壳
